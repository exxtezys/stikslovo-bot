# 🎯 StickerBot — Поиск стикеров по словам в Telegram

Бот, который позволяет находить стикеры по словам (как в TikTok).
Отправляешь боту стикер → он запоминает эмодзи и привязывает слова.
В любом чате пишешь `@botname слово` → бот показывает подходящие стикеры.

---

## 🚀 Быстрый старт (для обычного пользователя)

### 1. Получи токен бота

1. Открой Telegram, найди **[@BotFather](https://t.me/BotFather)**
2. Отправь команду `/newbot`
3. Придумай имя (например: `Мои Стикеры`)
4. Придумай username (например: `moistikeribot` — должно заканчиваться на `bot`)
5. Скопируй токен (выглядит как `123456:ABCdef...`)

> ⚠️ **Важно**: потом отправь BotFather команду `/setinline`, выбери своего бота, и напиши любое пояснение (например: `Поиск стикеров`). Это включит inline-режим.

### 2. Установи Python

Если Python не установлен:
1. Скачай с [python.org](https://www.python.org/downloads/) (версия 3.10+)
2. При установке **обязательно поставь галочку** ✅ «Add Python to PATH»

### 3. Настрой бота

1. В папке `sticker-bot` создай файл `.env`
2. Запиши в него свой токен:

```
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

### 4. Запусти

Дважды кликни **`run.bat`** — бот запустится.

> Первый запуск установит зависимости (30 секунд). Окно не закрывай — бот работает, пока окно открыто.

## 📱 Как пользоваться (на iOS / любом устройстве)

### Добавить стикеры

1. Найди своего бота в Telegram, нажми **Start**
2. **Отправь боту стикер** (прямо из клавиатуры стикеров)
3. Бот запомнит его эмодзи и привяжет слова
4. **Повторяй** с любыми стикерами

### Импортировать целый стикер-пак

Отправь боту: `/import ИмяПака`

### Искать стикеры

В **любом чате** начни писать `@твой_бот` и слово — бот покажет стикеры.

Примеры:
- `@moistikeribot смех` → покажет стикеры с 😂
- `@moistikeribot любовь` → покажет стикеры с ❤️
- `@moistikeribot привет` → покажет стикеры с 👋

## 📊 Команды бота

| Команда | Что делает |
|---------|-----------|
| `/start` | Показать справку |
| `/mysets` | Сколько стикеров сохранено |
| `/remove` | Удалить последний стикер |
| `/clear` | Удалить ВСЕ стикеры |
| `/import НазваниеПака` | Импорт целого стикер-пака |

## 🛠 Для продвинутых

### Запуск на сервере (Linux)

```bash
cd sticker-bot
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
nano .env  # вставь BOT_TOKEN
python main.py
```

Для фонового режима используй `screen` или `systemd`.

## ❓ Частые вопросы

**Q: Бот не отвечает на @имя слово?**
A: Проверь, что ты отправил `/setinline` в BotFather. Без этого inline-режим не работает.

**Q: Стикеры не находятся по слову?**
A: У стикера должно быть эмодзи. Открой стикер в Telegram — если под ним нет эмодзи, бот не сможет его привязать.

**Q: Как добавить свои слова к эмодзи?**
A: Отредактируй файл `emoji_map.py` — добавь нужные слова в список для эмодзи.

**Q: Где хранятся данные?**
A: В файле `stickerbot.db` (SQLite) в папке бота. У каждого пользователя своя коллекция.
